### 一个让mc合成更真实的mod
目前准备实现：
- 工作台进行工具箱判定
- 合成加上制作进度条
- 物品切割的方法制作速度和工具锋利度有关
- 砂轮可以消耗物品耐久增加锋利度
- 木头类物品加工消耗工具箱里面切割类工具锋利度
- 石头同理，但是加上工具硬度判断并且锋利度消耗更多
- 切石机加一个锯片槽，机制同上，速度加倍，加上自动交互
- 粉碎类加入研钵、磨盘
- 研钵消耗耐久合成，磨盘有巨量耐久并可自动化
- 烧制类炉子加上点火器插槽，每次启动消耗打火石耐久
- 金属不能被熔炉燃烧
- 高炉合成表加入鼓风机
- 金属加工需铁砧或锻造台
- 粘合类合成需要消耗粘液球、浆糊或钉子
- 食物加工加入烧焦判定
- 有耐久物品根据玩家等级调整合成后最大耐久
